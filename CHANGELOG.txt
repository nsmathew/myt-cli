CHANGELOG
----------------------------------------------------------------
v0.7.1 - Released on 13-Apr-2026

-- Features --
* Notes: support markdown link format [Title](https://url) in add/modify
  commands in addition to the existing https://url [Title] format. Works
  with urlopen for listing and opening links
* Notes: &"https://url" shorthand now expands correctly — URLs containing
  ':' were previously passed through unexpanded due to a filter-check
  ordering bug in the shorthand preprocessor

-- Fixes --
* TUI: urlopen no longer hangs — it was dispatched synchronously on the
  main event loop thread, causing a deadlock when _tui_prompt_callback
  tried to schedule a dialog on the same blocked thread. It now runs in a
  background thread via a new PROMPT_COMMANDS set
* TUI: urlopen dialog now shows the actual URL strings as choices instead
  of bare index numbers
* TUI: urlopen result is shown in the status bar instead of the main view,
  keeping the task list intact and not corrupting the auto-refresh command
* TUI: browser noise (e.g. "Opening in existing browser session.") no
  longer appears in the command input area — stdout/stderr are suppressed
  at the file-descriptor level during webbrowser.open() in TUI mode
* Scoring: replace quadratic due-date formula (d²/due_sum) with
  exponential decay. Due score is now list-independent, cannot go negative,
  and correctly orders overdue > today > future (old base values 99 vs 99.5
  had this backwards). See commit message for full formula details
* Scoring: fix incep_diff_now using timedelta.seconds (wraps at 86 400 s)
  instead of total_seconds() — tasks older than 24 h had random inception
  scores

v0.7.0 - Released on 12-Apr-2026

-- Features --
* TUI: New shorthand tokens for recurrence and notes in add/modify commands
  - *value expands to -re (recurrence), e.g. *M for monthly
  - *value|end expands to -re and -en (recurrence with end date), e.g. *M|+365
  - *|end expands to -en only (amend end date on existing recurring task)
  - &"text" expands to -no (notes)
* TUI: F8 toggles a popup showing the full last status message, useful when the toolbar is too narrow to display it in full. Dismiss with F8 or Esc
* TUI: --help now appears in autocomplete suggestions for all commands

-- Fixes --
* TUI: Unrecognised filter tokens (e.g. 'modify xyz') were silently dropped and treated as 'no filter', causing a prompt to modify all tasks. They now produce a clear 'No valid filters recognised' message and abort
* TUI: exit_app was returning instead of raising SystemExit in TUI mode, so confirmation prompts the user declined were ignored and execution continued. Commands now stop correctly when the user declines a prompt
* Help text: shorthand reference table in 'add --help' was displaying as a single wrapped line due to Click paragraph collapsing. Fixed with \b marker

-- Other --
* README updated to cover TUI mode, shorthand syntax, keyboard shortcuts (F5/F6/F7/F8) and updated screenshots
* Help text for add and modify updated with full shorthand reference, context filter examples, and corrected typos

v0.6.0 - Released on 09-Apr-2026

-- Fixes --
* Recurring tasks: fix hide date offset corruption on modify-all — series now recreates from the base task instead of the filtered derived instance, so the result is independent of which instance was picked
* Recurring tasks: preserve hide/due offset when only due is changed on modify-all (base task seed is shifted proportionally)
* Recurring tasks: reject absolute and today-relative (+N) hide dates on modify-all with a clear message — a due-relative offset (e.g. -3) or 'clr' is required since each instance has its own due date
* Recurring tasks: reject clearing due (clr) on modify-all to prevent crash in instance generation
* Recurring tasks: prompt to optionally shift hide by the same delta when changing due on a single instance (modify-this), defaulting to no

-- Other --
* TUI: Disable mouse support to restore native terminal text selection (click-drag to copy) across table and non-table views
* Added 19 regression and fix tests covering recurring task hide date scenarios
